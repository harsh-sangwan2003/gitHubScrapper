## Tools Used

- NodeJs
- JavaScript
- xlsx
- request
- cheerio

## Requirements

Please install the following npm packages before running the script

- nodejs
- npm

To install the dependencies run:

```js
npm install dependencies
```

## Usage

The project is a Nodejs script that can be run from the command line.
Type 

```js
node main.js
``` 
to run the script
